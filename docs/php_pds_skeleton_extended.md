This is a extended package skeleton inspired by the [Standard PHP package skeleton](https://github.com/php-pds/skeleton). 

## Standard PHP package skeleton.

```
bin/                # command-line executables
config/             # configuration files
docs/               # documentation and examples
public/             # web server files
src/                # PHP source code
resources/          # other resource files
tests/              # test code
vendor/             # reserved for package managers
CHANGELOG           # a log of changes between releases
CONTRIBUTING        # guidelines for contributors
LICENSE             # licensing information
README              # information about the package itself
```

## Extended PHP package skeleton
```
build/              # build and artifact files
examples/           # example files
lib/                # libraries
migrations/         # database migration files
tmp/                # local temp files
tmp/cache/          # local cache files
tmp/log/            # local log files

public/images/      # image files
public/js/          # java-script files
public/css/         # css files
public/icons/       # ico file
public/fonts/       # web fonts
public/less/        # less files
public/scss/        # scss files

src/Controller/     # Controller classes
src/Model/          # Model classes
src/View/           # Template files
src/Service/        # Model: Service layer classes
src/Util/           # Util classes and functions
src/Helper/         # Helper classes and functions
```
